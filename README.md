Overview of Project


**Activity 1** (Student): Creates a Student class (a blueprint) and uses it to make two Student objects, each with its own name, age, and grade.

**Activity 2** (Encapsulation): Modifies the Student class to hide its data (private) and adds public getter/setter methods for controlled access.

**Activity 3** (Vehicle): Creates a Vehicle class with a method (accelerate) that updates the object's own speed attribute.

**Activity 4** (Bank Account): Creates a BankAccount class with methods that use logic (like checking for sufficient funds) before changing the balance.

**Activity 5** (Circle): Creates a Circle class that uses Java's built-in Math library to calculate and return the circle's area and circumference.
